# 钱包

# 为什么要充值

企业在项目开始之前，需要对账户进行充值，充值完成后，专家才可进行工作。

# 充值步骤

目前平台只提供线下打款方式，其他在线充值方式敬请期待。具体步骤如下：

  1. 登录企业账号
  2. 打开导航【钱包】模块，点击充值按钮
  3. 线下完成打款后，在线录入打款信息，提交后等待平台确认，确认时间通常为1个工作日内
  4. 平台确认后，即可充值成功。

# 钱包余额

钱包余额是动态的，具体规则如下：

  • 余额增加：企业充值成功
  • 余额减少：企业确认专家日志后，即时扣款

您可以查看钱包页面下方的记录核对账单信息。

# 申请发票

企业可在线申请开具发票，关于申请发票的几个说明：

  • 发票金额：企业可开票金额，不是充值金额，而是累计的支出金额，即支付给专家的总费用
  • 发票类型：增值税普通发票和增值税专用发票两种类型，企业可按需申请
  • 领取方式：平台将以快递方式邮寄发票

具体的申请步骤如下：

  1. 登录企业账号
  2. 在导航的【个人设置】模块，填写【收件信息】
  3. 打开【钱包】模块，点击【申请发票】
  4. 按要求填写开票信息，提交后等待平台处理
  5. 平台处理完成后，将会展示快递信息，请耐心等待，签收快递后即可全部完成
