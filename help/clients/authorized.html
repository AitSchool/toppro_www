# 企业认证

# 认证理由

  • Toppro将优先为认证企业提更好的服务
  • 企业认证成功后，可成为Toppro的合作伙伴
  • 企业认证成功后，发布项目并支出后，可在Toppro申请开具发票

# 认证标准

企业认证时，Toppro主要核对填写的信息真实无误，以后开具发票等都建立在认证的基础上，请务必准确填写。

# 认证流程

  1. 登录企业账号
  2. 查看【个人设置】-【企业认证】，填写认证资料并提交
  3. 等待Toppro审核（通常为1个工作日内），审核通过即可认证成功
