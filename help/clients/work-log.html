# 日志管理

# 什么是日志

开始为企业服务的专家，每天都会提交当日的日志，用来说明当日完成的工作内容、工作时长等。专家提交日志后，Toppro平台会首线审核，审核通过后才会展示给企业查看。企业进行最终审核。

# 为什么要进行日志确认

企业需要对专家的日志进行确认，主要目的如下：

  • 为了确认专家的工作成果和工作量
  • 为了掌握项目的实时进展
  • 确认日志后，Toppro平台会即时进行服务结算，企业即时扣款。

# 日志确认步骤

  1. 登录企业账号
  2. 查看导航【日志管理】模块，点击日志列表的【确认】按钮，即可确认成功。如果逐个确认操作麻烦，可选择【全部确认】，即可全部确认成功。

# 申请仲裁

如果您对专家提交的工时不满意，可以申请仲裁。申请记录将会记录到您的历史行为中，请谨慎处理。 申请仲裁步骤如下：

  1. 打开专家日志详情，点击【仲裁图标】



  2. 填写【申请工时】和【申请理由】后，提交。



  3. 平台将在3个工作日内，与您和对应专家沟通，并给出处理结果。
  4. 平台给出处理结果后，您将收到消息通知，可查看处理结果。
