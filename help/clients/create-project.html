# 新增项目

# 什么是新增项目

企业可以发布项目需求，Toppro将根据您的需求为您寻找合适的人才。

# 新增项目步骤

  1. 登录企业账号，进入左侧导航【项目管理】模块，点击【新增项目】



  2. 填写项目基本信息和人才需求
    • 基本信息：描述您的项目的基本情况
    • 人才需求：描述您需要的人才类型和要求



  3. 提交后，即可创建成功

# 新增项目后服务

新增项目后，Toppro工作人员在将会主动与您联系，了解您的项目的详细情况，然后为您推荐合适的人才，推荐成功后，即可雇佣人才，人才开始为您服务。
