# 专家认证

# 认证理由

  • 专家认证成功后，成为Toppro平台签约专家，参与众多高质量的项目。
  • 专家认证成功后，可获得Toppro平台相应的职位等级评定和更有竞争力的回报。
  • 专家认证成功后，可获取Toppro专家福利，如：可定期参与组织沙龙聚会，可在开放的办公空间工作等。

# 认证标准

专家认证的职级评定标准，由Toppro平台综合知名互联网公司职级评定模型后建立的职级标准，具体规则和要求暂不公示，等级分类如下：

  • PD1：初级
  • PD2：中级
  • PD3：高级
  • PD4：资深
  • PD5：专家
  • PD6：研究员

# 认证流程

  1. 未认证时，注册或登录后会自动跳转认证信息填写页面
  2. 在认证信息填写页面上，填写完认证资料后进行提交
  3. 请耐心等待，Toppro会主动与您沟通（正常在1个工作日内），沟通完成并符合标准后，即可认证成功

# 认证失败

专家提交资料申请认证后，经过与Toppro工作人员沟通后，如果不符合Toppro专家要求的标准，将不予认证，即认证失败。认证失败后，平台将不会让专家参与平台项目，也不支持再次认证，非常感谢您的支持。认证失败的如下但不限于以下几条：

  • 工作年限较短的
  • 工作能力不足的
  • 个人及工作信誉较差的
  • 有其他不符合Toppro要求的
