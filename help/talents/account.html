# 个人资料

专家账号登陆后，可以到个人设置修改自己的个人资料，包含头像、昵称、所在地区等

# 认证信息

在认证信息中，可以查看自己的职级和服务价格等信息，认证的经历信息暂不可修改

# 提现账号

认证成功的专家，在平台需要设置提现账号，每月结算报酬申请提现时需要用到。
您可以设置自动提现，这样每月1号时平台将自动为您生成提现记录，然后进行结算处理
