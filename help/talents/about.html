# Toppro介绍

Toppro是一个企业级人才资源服务平台。
Toppro致力于构建优秀人才网络，让优秀的人才有更好的回报，更好的项目和工作方式。

# 专家账号

专家账号专门服务于专家，让您可以在Toppro平台自由工作，在这里您可以：

  • 进行专家认证，Toppro将会对您进行等级评估和服务价格确认，认证成功后，您将获得更好的服务
  • 在有合适的项目时，Toppro将会与您沟通，确认服务意向后，即可参与项目、开始工作

详细介绍，请查看官网专家招募的介绍
