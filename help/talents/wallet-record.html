# 账户余额

专家参与项目后，可实时查看钱包中的账户余额，账户余额规则如下：

  • 余额增加：平台审核通过日志后，余额即相应增加
  • 余额减少：专家申请提现，余额即相应减少

专家可查看钱包页面下方的收支明细，进行账单核对。

# 申请提现

  1. 提现限制:您在任何时间都可以申请提现,提现处理将在下月1号下个工作日完成.
  2. 到账时间:每月1号为提现处理截止日期,平台将在下一个工作日内完成打款.

提现流程：

  1. 在【个人设置】-【提现账号】里，添加提现账号。
  2. 在钱包页面【申请提现】。
  3. 等待平台处理，处理完成后即可提现成功。

# 税金计算

按照国家法律规定，劳务报酬收入所缴纳相应税金，计算方式如下：

收入类型：劳务报酬所得
计算方式：超额累进
劳务报酬 = 税前金额
应纳税所得额 = 需要纳税的部分

应纳税所得额 = 劳务报酬（少于4000元） - 800元
应纳税所得额 = 劳务报酬（超过4000元） × （1 - 20%）

应纳税额 = 应纳税所得额 × 适用税率 - 速算扣除数

含税应纳税所得额不超过2W，税率20%，速算扣除数0
含税应纳税所得额超过2W不超过5W，税率30%，速算扣除数2000
含税应纳税所得额超过5W，税率40%，速算扣除数7000
