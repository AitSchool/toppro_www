# 产品经理

产品经理参与项目完成后，需提交资料如下：

  • 原型源文件（必须）
  • PRD文件（必须）
  • 核心业务流程图（必须）
  • UE图（必须）

# 设计师

设计师参与项目完成后，需提交资料如下：

  • 设计稿源文件（必须）
  • 设计稿标注图（必须）
  • UX交互图（必须）
  • 设计规范（必须）

# 开发工程师

开发工程师参与项目完成后，需提交资料如下

  • 代码源文件（必须）
  • 接口文档（必须）
  • 技术文档（必须）
