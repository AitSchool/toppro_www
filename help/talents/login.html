# 账号登录

已注册账号才能登录，目前仅支持手机号登录，您可以采用密码登录和手机验证码登录两种登录方式。

# 密码登录

  1. 打开登录页
  2. 输入您注册的手机号和密码，点击【登录】，即可成功

# 手机验证码登录

  1. 打开登录页
  2. 点击【手机验证码登录】
  3. 输入您注册的手机号，获取验证码并准确填写，点击【登录】，即可成功

# 忘记密码

如果您忘记注册账号的密码，我们为您提供两种解决方案：

  1. 采用手机验证码登录：填写您注册的手机号并获取短信验证码，即可登录成功，登录成功后在【个人设置】里修改密码并牢记。
  2. 如果您手机不在身边或者手机号已注销，请联系我们，我们的工作人员将会帮您解决登录问题。联系方式：010-86469068，hey@weilaigongzuo.com

# 修改密码

登录账号后，进入【个人设置】—【修改密码】，输入手机验证码和新密码，提交后即可修改成功。

# 修改手机号

如果您需要修改账户手机号，请联系我们，我们的工作人员将会帮您修改。联系方式：010-86469068，hey@weilaigongzuo.com
