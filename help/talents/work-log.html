# 日志管理

# 为什么要写日志

专家参与项目时，需要及时记录当日工作的完成情况和工作时长，计算报酬时以日志为依据

# 日志规范

专家撰写日志时，需遵守以下规范：

  1. 格式规范：按1、2、3格式填写，每条只填写一种工作内容
  2. 内容规范：
    1. 开发完成的内容。如：获取视频详情API。
    2. 与企业沟通项目需求。如：电话会议沟通接下来的开发任务。
    3. 文档撰写。如：项目说明文档。接口文档。
  3. 审核规范：
    • 工作时长：工作时间段与工作时长必须相符合
    • 工作内容：完成的工作量应与工作时长对应





# 如何撰写日志

有两个撰写日志入口：

  1. 点击【工作日志】，点击【撰写日志】，即可；
  2. 点击【我的项目】，找到要添加日志的项目，点击【撰写日志】，即可。

撰写日志内容：

  1. 选择所属项目；
  2. 填写今日完成、明日计划和备注信息；
  3. 选择工作日期：如果是当天，不用选择就可以；如果想补之前的日志，把工作日期选择下即可；
  4. 选择工作时间段：添加多个时段，比如9：00-10：00，15：00-16：00，提交即可。

小贴士：跨凌晨日志如何填写，举个例子：比如10月11日，从晚上22：00工作到凌晨1点，撰写具体如下：

  1. 选择工作日期为2018-10-11；
  2. 添加工作时间段，选择为2018-10-11 22：00 ~ 2018-10-12 01：00





# 日志审核流程

  • 专家提交当日工作日志
  • Toppro平台对专家提交的日志进行审核
    • 审核通过时：通过后，专家即可获得相应报酬
    • 审核不通过时：平台会说明不通过的原因，专家需对日志进行修改后，再次提交，再由平台进行审核
