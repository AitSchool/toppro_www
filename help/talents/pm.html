# 项目管理

# 参与项目

专家认证成功后，有合适的项目时，Toppro将会与您沟通项目需求、确认您的工作意向，请耐心等待。确认成功后，专家即可参与项目

# 管理项目

专家参与项目后，可查看项目进度、个人工作情况、撰写工作日志等。工作流程如下：

  1. 登陆专家账号
  2. 打开【我的项目】页面，可查看参与的项目情况与个人的工作进展情况，包括工作时长、工作收入等
  3. 当日工作完成后，点击【撰写日志】，填写今天的工作情况，等待平台审核。

# 退出项目

专家退出项目有三种情况，分别如下：

  • 项目工作已完成：项目正常完成时，即可退出项目，不再撰写日志。
  • 企业主动结束项目：因企业原因结束项目，专家工作即时停止，退出项目。
  • 专家个人原因中途退出：因专家个人原因需要中途退出，专家需提前一周跟Toppro工作人员联系并说明情况，然后处理后交接工作后即可退出项目。
